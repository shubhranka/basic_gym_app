# Gym App

A modern, high-performance mobile application built with **React Native** and **Expo**, designed to streamline the gym experience for both members and administrators.

### For Members
- **Member Dashboard**: Personalized hub displaying membership status and profile details.
- **Digital Access**: Integrated QR code generation for easy gym entry.
- **Benefits & Pricing**: Clear overview of membership perks and pricing tiers.
- **Modern UI**: Sleek dark mode design with vibrant lime accents.

### For Administrators
- **Admin Panel**: Secure area for staff to manage gym operations.
- **QR Scanner**: Built-in camera integration to scan member QR codes for check-ins.
- **Member Management**: Create, view, and manage member profiles.
- **Kiosk Mode**: Dedicated mode for front-desk check-ins.

## ğŸƒ Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:
```bash
npx expo start
```

- **Run on iOS**: Press `i` in the terminal (requires Xcode/Simulator).
- **Run on Android**: Press `a` in the terminal (requires Android Studio/Emulator).
- **Run on Device**: Scan the QR code with the Expo Go app.
